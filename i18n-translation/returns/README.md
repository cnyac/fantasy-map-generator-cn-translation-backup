# 📥 翻译回收箱

**把外包 AI 翻译好的 JSON 文件，直接拖到这个文件夹里就行**（文件名随便叫什么都可以，只要是 `.json`）。

放好之后，运行：

```
npm run i18n:merge
```

它会自动：
1. **校验**这里所有 JSON（检查有没有漏译、占位符丢失、术语表不一致等，有问题会列出来让你退回 AI 改）；
2. 校验通过后**回填**进运行时字典；
3. 把处理完的文件挪进 `_archive/` 存档（避免重复回填）。

然后浏览器打开 `http://localhost:5173/Fantasy-Map-Generator/?locale=zh-CN` 刷新，就能看到新翻译生效。

> 只想先检查不回填，用 `npm run i18n:validate`。
